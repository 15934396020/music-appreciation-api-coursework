Review CRUD demonstrates complete lifecycle management

Create, read, update, and delete reviews with automatic rating recalculation

The Reviews resource is the primary CRUD entity. Creating a review validates the track exists and enforces rating (1–5) and comment constraints. Updates support partial modification. Deletion returns 204 No Content. After every create, update, or delete operation, a helper function automatically recalculates and persists the track's average rating.

Endpoint Description
POST /reviews Create a review (validates track, enforces constraints, returns 201)
GET /reviews?track_id=5&min_rating=4 Filter reviews with pagination
PUT /reviews/{id} Partial update (only provided fields change)
DELETE /reviews/{id} Remove review, recalculate track average (returns 204)
